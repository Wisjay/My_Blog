My First Coding Experience

My first coding experience was both exciting and challenging. Before I started, I thought programming was only for people with exceptional technical skills. However, once I wrote my first lines of code, I realized that coding is more about solving problems step by step than memorizing complex commands.

At first, I struggled to understand basic concepts and i made simple errors, like forgetting a semicolon or misspelling a command which caused my program to fail. These mistakes were frustrating, but they taught me the importance of paying attention to detail and reading error messages carefully.

One of the most rewarding moments was when I created my first simple web page "Hello World" using HTML, then proceded to write a brife introduction about myslef and then my gold of learning UI/UX design. Seeing the text appear exactly as I had designed them gave me a great sense of accomplishment. It showed me that with patience and practice, I could turn ideas into something real and interactive.

Learning the basics: HTML and CSS

Learning the basics of HTML and CSS has been an exciting and rewarding step in my journey toward becoming a programmer. At the beginning, I knew very little about how websites were created, but as I started studying these technologies through the EHA Academy 2026 Web and Software Development (Design Phase) Cohort, I realised that every webpage is built on a structured foundation and enhanced with creative styling.

I began by learning HTML (HyperText Markup Language), which is the standard language used to structure web pages. I discovered how to create headings, paragraphs, lists, tables, images, and hyperlinks. I also learned the importance of semantic HTML elements such as header, nav, main, section, article, and footer, which make web pages more organised, accessible, and easier to understand.
After becoming familiar with HTML, I moved on to CSS (Cascading Style Sheets), which is used to control the appearance and layout of web pages. I learned how to change text styles, colors, backgrounds, margins, padding, borders, and spacing. I also explored different selectors, the box model, and how to use classes and IDs to apply styles efficiently. Seeing a plain HTML page transform into a visually appealing website through CSS was both satisfying and motivating.

One of the most valuable lessons I learned was the importance of separating content from presentation. HTML provides the structure of a webpage, while CSS handles its design. This separation makes websites easier to maintain, update, and scale as they become more complex.
Throughout my learning process, I completed several hands-on exercises, including creating simple web pages, navigation menus, tables, forms, and styled blog layouts. These practical activities helped reinforce my understanding of both HTML and CSS and improved my confidence in writing clean, organised code.

Building My First Projects

The first project I created was a Travel Webpage. The goal was to design a webpage that introduces visitors to an exciting travel destination.
I used HTML to structure the content by adding headings, paragraphs, images, and lists to showcase popular attractions, local culture, and travel tips. CSS was used to improve the appearance of the webpage. I experimented with different fonts, colors, background images, spacing, and text alignment to make the page attractive and easy to read. I also learned how to style navigation links and organize the content into sections, giving the webpage a professional look.

My second project was a Recipe Book, where I created a webpage containing different recipes. Each recipe included the name of the dish, a list of ingredients, preparation steps, cooking time, and serving size. I also added a table of contents with internal links so users could quickly navigate to any recipe on the page. This project gave me more experience using HTML elements such as headings, lists, tables, images, and anchor tags. I also used CSS text properties to customize the font family, font size, font style, line height, and text alignment, making the content more readable and visually appealing.

z While working on these projects, I discovered the importance of planning a webpage before writing code. I also learned how CSS can completely transform the appearance of a simple HTML document without changing its content. Creating these projects has taught me how to organize information clearly and create a user-friendly layout. I also gained confidence in using internal navigation, which improves the user experience on longer webpages.

The Journey Continues

Every learning journey comes with its own set of challenges, and mine was no different. At first, remembering HTML tags and figuring out how different CSS properties worked together felt overwhelming. There were times when a small mistake, like forgetting to close a tag or using the wrong CSS selector, would completely change how my webpage looked. It was frustrating at times, but each challenge became an opportunity to learn something new.
Instead of giving up, I kept practicing, experimenting with different code snippets, and learning from my mistakes. Gradually, the concepts that once seemed difficult became much easier to understand. Every project I completed strengthened my problem-solving skills, improved my attention to detail, and gave me the confidence to try more creative designs and layouts. It showed me that coding is a skill that improves with patience and consistent practice and has boosted my curiosity to keep building better and more interactive websites.

Looking ahead, I am excited to continue expanding my skills by learning JavaScript, responsive web design, and modern frameworks such as React.
My goal is to build websites that are not only visually appealing but also interactive, accessible, and user-friendly. I also aspire to develop real-world projects that solve practical problems, contribute to open-source communities, and eventually become a full-stack web developer capable of creating complete web applications from start to finish.

This is only the beginning of my programming journey, and I know there is still so much to learn. Every new concept, every project, and every challenge brings me one step closer to becoming a skilled developer. I look forward to continuing this journey, embracing new technologies, and turning innovative ideas into meaningful digital solutions.
